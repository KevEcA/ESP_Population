Manual de uso

1. Introducción

Este programa tiene como objetivo analizar la población viva y muerta de bombas ESP. En las gráficas de barras se muestra la cantidad de pozos, mientras que en las gráficas de box and whisker se visualiza la distribución del RL. El usuario puede agrupar la población en diferentes bins de RL. Finalmente, la última sección utiliza el método Kaplan–Meier (KM).

2. Input esperado

  • Well_ID: Identificador del pozo.
  • Run_Date: Fecha de arranque (formato día-mes-año, ej. 15-Sep-23).
  • Stop_Date: Fecha de paro. Si se deja en blanco, se asume que la bomba sigue operando.
  • State: 1 = pozo fallado, 0 = pozo no fallado o censurado.
  • Cause: Categorías permitidas (escribir exactamente):
    • Fail
    • Manual off
    • Tbg/Csg

Estas categorías se usan para diferenciar causas de paro y para definir si un pozo se considera fallado o censurado.

3. Conceptos

Bins: Intervalos de RL que agrupan la población en rangos definidos por el usuario.

Failed pump: Bomba que ha fallado (State=1).

Censored pump: Bomba que no ha fallado pero se detuvo por otra razón o sigue activa.

Kaplan–Meier (KM): Método estadístico robusto para estimar la probabilidad de supervivencia en poblaciones con datos censurados.

Box and whisker plot: La línea central representa la mediana, la caja el rango intercuartílico (Q1–Q3), los whiskers los valores dentro de 1.5 IQR y los puntos fuera son outliers.

Intervalos de confianza: Se calculan a partir de la varianza estimada de la función de supervivencia (ej. método Greenwood). Representan el rango donde se espera que esté la verdadera curva con un nivel de confianza (ej. 95%).

User Manual

1. Introduction

This program analyzes the active and failed ESP pump population. Bar charts show the number of wells, while box and whisker plots display the RL distribution. Users can group the population into different RL bins. Finally, the last section applies the Kaplan–Meier (KM) method.

2. Expected Input

  • Well_ID: Well identifier.
  • Run_Date: Start date (format day-month-year, e.g., 15-Sep-23).
  • Stop_Date: Stop date. If left blank, the pump is assumed to still be running.
  • State: 1 = failed well, 0 = censored or still active.
  • Cause: Allowed categories (must be written exactly):
    • Fail
    • Manual off
    • Tbg/Csg

These categories are used to distinguish reasons for stopping and to classify wells as failed or censored.

3. Concepts

Bins: RL intervals grouping the population into ranges defined by the user.

Failed pump: Pump that has failed (State=1).

Censored pump: Pump that has not failed but was stopped for another reason or is still active.

Kaplan–Meier (KM): A robust statistical method to estimate survival probability in populations with censored data.

Box and whisker plot: The central line represents the median, the box the interquartile range (Q1–Q3), the whiskers values within 1.5 IQR, and points outside are outliers.

Confidence intervals: Derived from the variance of the survival function (e.g., Greenwood’s formula). They represent the range where the true curve is expected to lie with a given confidence level (e.g., 95%).
